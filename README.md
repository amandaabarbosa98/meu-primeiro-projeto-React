# My Daily Habits 🗓️

Este é o meu primeiro projeto desenvolvido com **React**. O objetivo da aplicação é auxiliar no gerenciamento de hábitos diários, permitindo que o usuário acompanhe suas rotinas de forma simples e intuitiva.

O projeto faz parte da minha trilha de aprendizado em desenvolvimento front-end, explorando conceitos como componentes, hooks e estados.

---

## 🚀 Tecnologias Utilizadas

* **React.js**: Biblioteca principal para construção da interface.
* **JavaScript (ES6+)**: Lógica de programação.
* **CSS3**: Estilização e layout responsivo.
* **HTML5**: Estrutura semântica.

---

## 🛠️ Funcionalidades (Em desenvolvimento)

* Visualização e listagem de hábitos diários.
* Controle de progresso das atividades.
* Interface amigável e limpa.
